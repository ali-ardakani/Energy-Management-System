## Getting Started

<!-- Setup: -->
<!-- Docker -->
### Docker
1. Clone the repo
   ```sh
   git clone
    ```
2. Build and run the docker container
   ```sh
   docker-compose up --build
   ```
3. Open the application in your browser
   ```sh
    http://0.0.0.0:8000/
    ```
